# 🍍 Spongebob MBR Jokes Virus (Prank Script)

This is a harmless prank script created using **AutoHotkey v2**. It simulates a fake MBR "corruption" screen with glitch effects and Spongebob music.

### ⚠️ DISCLAIMER
**THIS IS NOT A REAL VIRUS.** This script does NOT modify your Master Boot Record (MBR), delete files, or damage your hardware. It is created strictly for educational and entertainment purposes.

### 🚀 HOW IT WORKS
1. **Auth:** Requires a password to start (Default: `12`).
2. **Chaos Phase:** 5 colorful windows appear, stay still for 5s, then bounce rapidly across the screen for another 5s.
3. **Sound:** Plays a looping error sound, then switches to Spongebob music.
4. **MBR Screen:** A black screen appears with a custom typing effect.
5. **Panic Control:** Press `ESC` at any time to instantly kill the script and restore your cursor.

### 🛠️ REQUIREMENTS
- Windows OS
- [AutoHotkey v2](https://www.autohotkey.com/)
- Audio files: `erro.mp3` and `spongebob-music_-grass-skirt-chase-1.mp3` (Must be in the same folder).
